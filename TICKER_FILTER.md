# TICKER_FILTER.md - False Ticker Collision List

This file contains common English words, corporate acronyms, and Reddit slang that frequently collide with valid US equity tickers. These should be excluded during the `extract_tickers` pipeline phase unless explicitly prefixed with a `$`.

## Common English Words
A, I, IT, ON, IN, TO, DO, BE, UP, ALL, OUT, FOR, ARE, ANY, HAS, WAS, NOW, SEE, DAY, BUY, RUN, BIG, MAN, CAN, NEW, NOW, ONE, TWO, SIX, TEN, CAR, JOB, PAY, TAX, JOSE

## Corporate & Financial Acronyms
CEO, CFO, CTO, COO, DD, EPS, ROI, YTD, SEC, FED, GDP, ATH, OTC, IPO, PNL, PR, HR, LLC, INC

## Reddit / WSB Slang
YOLO, FOMO, LFG, WSB, MOON, HOLD, PUMP, DUMP, APE, APES, BULL, BEAR, GUH, TEND, DFV, RH

## General / Macro Themes
USA, UK, EU, AI, EV, AR, VR, PC, TV, ETF
